Semantic elements

HTML markup What is it? How to use it? Useful tips
<title> The title element is telling us the title of the webpage. The content between the tags is shown in the browser's title bar or tab and will be the default if a user adds the page to their favourite. The title will also we shown in as the title for a page in a search engine result. The title element should go in the <head> element in the HTML. It does not go in the <body> of the document.
<header> The header element is a container for introductory content such as logo, company name, search feature, headings and navigational links. It usually works as a banner for the page and is re-used globally across the website. However, it can also be used within an article or a section to represent the introduction to that particular piece of content. The header usually goes in the top of the page, unless it is nested within an article or a section - in which case it goes in the top of that element.
